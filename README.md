Welcome to Labyrinth
Owner: James Suminski
Labyrinth is a quote-capturing app. This app is purely a passion project and is not meant to be monetized.

Labyrinth allows for quote browsing, quote favoriting, notification customization, quote categorization, and optional CSV imports. 

All quotes are gathered and approved by the app owner. Quotes are sourced from family and friends. 

Tech stack: Swift UI via Macincloud, Swift Data, IOS 17+, and Github.
